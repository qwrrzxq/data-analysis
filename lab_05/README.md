# Лабораторна робота №5

## Наука про дані: обмін результатами та початковий аналіз

### Опис завдання

Streamlit веб-додаток для інтерактивного аналізу VHI-даних (Vegetation Health Index) по областях України.

### Можливості

- Dropdown вибір індексу (VCI, TCI, VHI)
- Dropdown вибір області для аналізу
- Slider для вибору інтервалу тижнів та років
- Кнопка Reset для скидання фільтрів
- Три вкладки: таблиця, графік часового ряду, порівняння областей
- Checkbox сортування за зростанням/спаданням
- Інтерактивні графіки на Plotly

### Інструкція із запуску

**Вимоги:** Python 3.10+, pip

```bash
# 1. Створити віртуальне середовище
python -m venv venv

# 2. Активувати (Windows)
venv\Scripts\activate

# 3. Встановити залежності
pip install -r requirements.txt

# 4. Запустити Streamlit додаток
streamlit run lab5_streamlit.py
```

Додаток відкриється в браузері за адресою http://localhost:8501

### Структура інтерфейсу

| Елемент | Опис |
|---------|------|
| Dropdown "Індекс" | Вибір VCI, TCI або VHI |
| Dropdown "Область" | Вибір області для аналізу |
| Slider "Тижні" | Фільтрація за номером тижня (1–52) |
| Slider "Роки" | Фільтрація за роками |
| Checkbox "Сортування" | Сортування за зростанням/спаданням |
| Кнопка "Reset" | Скидання всіх фільтрів |
| Вкладка "Таблиця" | Відфільтровані дані з статистикою |
| Вкладка "Графік" | Часовий ряд обраного індексу |
| Вкладка "Порівняння" | Барчарт порівняння всіх областей |
